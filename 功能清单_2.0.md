这是一个为“跨国公司智能仓库管理系统”量身定制的详细功能清单。我们基于此清单进行开发。

### 📋 跨国公司智能仓库管理系统 - 功能清单 

**项目目标**：开发一个B/S架构的智能仓库管理系统，实现全球多仓库的数字化管理，核心亮点包括：库存可视化、智能货位推荐、多维度数据统计。

**系统用户角色**：
*   **仓库管理员**：系统主要使用者。

---

### 🧩 模块一：仪表盘 (Dashboard) - 【核心亮点】

| 功能点   | 功能描述                                                     | 优先级 |
| :------- | :----------------------------------------------------------- | :----- |
| **D-01** | **数据概览卡片**：展示关键数据，如：总库存品种数、总库存价值、待处理出入库单数量。 | 高     |
| **D-02** | **仓库库存分布饼图**：使用ECharts绘制图表，直观展示各仓库的库存量占比。 | 高     |
| **D-03** | **低库存预警看板**：以一个表格形式，高亮显示库存数量低于安全库存水平的物料。 | 高     |
| **D-04** | **近期出入库趋势图**：使用ECharts折线图，展示近30天的入库和出库数量趋势。 | 中     |

---

### 🧩 模块二：基础数据管理 (Master Data Management)

| 功能点   | 功能描述                                                     | 优先级 |
| :------- | :----------------------------------------------------------- | :----- |
| **M-01** | **仓库信息管理**                                             | 高     |
|          | - 列表显示所有仓库信息（仓库编码、名称、地址、本位币）。     |        |
|          | - 支持新增、修改、删除（逻辑删除）仓库。                     |        |
| **M-02** | **货位管理**                                                 | 高     |
|          | - 列表显示指定仓库下的所有货位（货位编码、所属仓库、状态：空闲/占用）。 |        |
|          | - 支持新增、修改、删除货位。                                 |        |
| **M-03** | **物料主数据管理**                                           | 高     |
|          | - 列表显示所有物料信息（SKU、名称、分类、规格、尺寸、重量、**安全库存**）。 |        |
|          | - 支持新增、修改、删除物料。                                 |        |
|          | - 支持按物料名称/SKU/分类进行搜索。                          |        |

---

### 🧩 模块三：库存管理 (Inventory Management) - 【核心模块】

| 功能点   | 功能描述                                                     | 优先级 |
| :------- | :----------------------------------------------------------- | :----- |
| **I-01** | **全局库存查询**                                             | 高     |
|          | - 可按**仓库**、**物料**等多个条件组合查询库存。             |        |
|          | - 结果以表格显示（物料、仓库、货位、数量）。                 |        |
| **I-02** | **采购入库**                                                 | 高     |
|          | - 创建入库单，选择供应商、物料、数量。                       |        |
|          | - **【核心亮点】智能货位推荐**：系统根据物料尺寸和货位空闲状态，自动推荐最优上架货位。 |        |
|          | - 确认入库后，更新库存数量。                                 |        |
| **I-03** | **销售出库**                                                 | 高     |
|          | - 创建出库单，选择客户、物料、数量。                         |        |
|          | - 支持**FIFO（先进先出）** 出库策略。                        |        |
|          | - 确认出库后，更新库存数量。                                 |        |
| **I-04** | **库存盘点**                                                 | 中     |
|          | - 生成盘点任务，录入实际盘点数量，系统生成盘盈盘亏报告。     |        |

---

### 🧩 模块四：系统管理 (System Administration)

| 功能点   | 功能描述                                                     | 优先级 |
| :------- | :----------------------------------------------------------- | :----- |
| **S-01** | **用户登录**                                                 | 高     |
|          | - 简单的用户名/密码登录验证。在前端实现一个“模拟登录”，可以省去设计用户表、编写密码加密、会话管理等复杂后端逻辑。 |        |

---

### ✨ 核心亮点

1.  **智能货位推荐** (入库时)：体现“智能”，优化仓库空间利用。
2.  **可视化数据看板**：用ECharts图表让数据一目了然，**特别是新增的“低库存预警”**，这是一个非常实用且不复杂的业务亮点。

这个版本的功能清单更加务实，所有功能都紧密围绕“仓库库存”这一核心主题，技术实现路径清晰，非常适合你们四人团队在既定时间内完成并做出高质量。
