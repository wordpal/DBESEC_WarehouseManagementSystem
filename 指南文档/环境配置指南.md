🛠️ 常见问题解决

问题1：404错误

bash

\# 原因：项目未正确部署

\# 解决：

1\. Eclipse中：项目右键 → Properties → Project Facets

2\. 勾选：Dynamic Web Module

3\. 应用 → 重新部署

问题2：数据库连接失败

bash

\# 原因：MySQL未启动或密码错误

\# 解决：

1\. 检查MySQL服务是否运行：services.msc

2\. 验证密码：mysql -u root -p050320

3\. 修改DatabaseUtil.java中的连接信息

问题3：端口冲突

bash

\# 原因：8080端口被占用

\# 解决：

1\. 修改Tomcat端口：conf/server.xml

2\. 查找<Connector port="8080" → 改为8081

3\. 重启Tomcat

问题4：编码乱码

bash

\# 原因：字符编码不一致

\# 解决：

1\. C同学已配置QuickEncodingFilter

2\. 确保所有文件保存为UTF-8

3\. Eclipse设置：Window → Preferences → General → Workspace → UTF-8
